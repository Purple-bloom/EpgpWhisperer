# EPGP Helper Addon For EPGP WITH GP BIDDING
Opens a window sorting all bids received - has 6 bid "levels": ms high, ms mid, ms low, os high, os mid, os low (sorted in that order)

Supports a prio import and sorting in the format: character1:prio.prio;character2:prio.prio;character3,altName:prio.prio; etc. (example: Calmora,Calbara:279.746;Aramok,Calsyn:277.142;Felthorn:211.186;)

/pimp is used for the prio import and supports up to 50,000 character input. (big pastes will freeze the game for a couple seconds)

/pap will print all prios of all current raid members in the raid chat

best used with the epgp webserver + backend server here:
